== Description ==

The Connect plugin by StackCommerce connects your WordPress CMS to the StackCommerce Articles repository.

Top features:

* Secure, automated content creation and posting without leaving the WordPress dashboard.
* No development work required. Activate the plugin and connect.
* Control the status. Posts can be created as drafts, pending, or scheduled.
* Choose the author you want the post to be created by.
* Set categories and tags to associate with all posts.


== Installation ==

= From your WordPress dashboard =

1. Visit `Plugins > Add New`
1. Search for `StackCommerce Connect`
1. Activate StackCommerce Connect from your Plugins page.

= From WordPress.org =

1. Upload the extracted archive to `wp-content/plugins/` (or upload a zip through the WordPress admin)
1. Activate the plugin through the `Plugins` menu in WordPress

= Once Activated =

1. Get your Account ID & Secret Key from StackCommerce.
1. Go to the `StackCommerce` plugin settings and setup your basic settings.
1. Content will be created under the Posts section, where you will see the current status (draft, pending, scheduled, or published).
